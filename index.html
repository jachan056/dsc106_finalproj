What have you done so far?

We have created a GitHub repo and set up the web page that will host the interactive visualization with an index.html file. We built a cover page with animated rain effects that fits the topic of the project. We implemented the first D3 interactive visualization that shows the seasonal precipitation cycle from historical data. We have also brainstormed the other visualizations to include in the project.

What will be the most challenging part of your project to design and why?

The most challenging part of the project to design would be building the map to show changing global and regional precipitation levels. One of the challenges is how to transform the CMIP6 data files into something that D3 can process so we can apply the interactions on the map. Another challenge would be how to encode the precipitation level changes on the map so it is not messy yet still representative globally. We also want to design a hover interaction that lets users select specific regions and see the changes in that region, which could be quite tricky to implement.
